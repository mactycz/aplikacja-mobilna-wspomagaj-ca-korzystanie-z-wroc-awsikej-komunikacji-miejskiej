# aplikacja-mobilna-wspomagaj-ca-korzystanie-z-wroc-awsikej-komunikacji-miejskiej
Aplikacja stworzona w Visual Studio w środowisku Xamarin. Dane z transportu do pobrania jest ze strony https://www.wroclaw.pl/open-data/dataset/rozkladjazdytransportupublicznegoplik_data, a następnie należy je dodać do folderu assets.
Aplikacja pokazuje rozkład jazdy oraz znajduje najlepsze połączenie pomiędzy dwoma podanymi przystankami.
